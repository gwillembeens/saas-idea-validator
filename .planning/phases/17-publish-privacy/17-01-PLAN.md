# Plan 17-01: Backend — Visibility Column & API

## Objective

Add `is_public` column to `saved_results`, expose it in all existing history endpoints, and create the new `PATCH /api/history/:id/visibility` endpoint.

---

## Tasks

### Task 1 — DB Migration: Add `is_public` column

**File:** `server/db/schema.sql`

Follow Phase 16 precedent (niche column). Add idempotent ALTER TABLE:

```sql
ALTER TABLE IF EXISTS saved_results
  ADD COLUMN IF NOT EXISTS is_public BOOLEAN NOT NULL DEFAULT true;
```

Place after the niche column migration. No backfill needed — `DEFAULT true` covers all existing rows automatically.

---

### Task 2 — `saveResultRoute`: Add `is_public` to response

**File:** `server/routes/history.js` (lines 57–61)

The INSERT does not set `is_public` (DB default handles it). Add `is_public: true` to the response statically — no DB round-trip needed since new rows always default to `true`.

Before:
```js
res.status(201).json({
  id: result.id,
  title: result.title,
  createdAt: result.created_at,
})
```

After:
```js
res.status(201).json({
  id: result.id,
  title: result.title,
  is_public: true,
  createdAt: result.created_at,
})
```

### Task 5 — Add `updateVisibilityRoute` function

**File:** `server/routes/history.js` (append after `updateTitleRoute`)

Follow the exact `updateTitleRoute` pattern (auth → validate → ownership check → UPDATE RETURNING → response):

```js
// PATCH /api/history/:id/visibility — toggle public/private with ownership check
export async function updateVisibilityRoute(req, res) {
  if (!req.user) {
    return res.status(401).json({ error: 'Unauthorized' })
  }

  const { id } = req.params
  const { is_public } = req.body

  if (typeof is_public !== 'boolean') {
    return res.status(400).json({ error: 'is_public must be a boolean' })
  }

  try {
    const { rows } = await pool.query(
      'SELECT user_id FROM saved_results WHERE id = $1 AND deleted_at IS NULL',
      [id]
    )

    if (!rows.length) {
      return res.status(404).json({ error: 'Result not found' })
    }

    if (rows[0].user_id !== req.user.id) {
      return res.status(403).json({ error: 'Forbidden' })
    }

    const updateRows = await pool.query(
      'UPDATE saved_results SET is_public = $1, updated_at = now() WHERE id = $2 RETURNING id, is_public',
      [is_public, id]
    )

    res.json({
      id: updateRows.rows[0].id,
      is_public: updateRows.rows[0].is_public,
    })
  } catch (err) {
    console.error('Update visibility error:', err)
    res.status(500).json({ error: 'Failed to update visibility' })
  }
}
```

## Verification

- `POST /api/history` response includes `is_public: true`
- `GET /api/history` items each include `is_public` field
- `GET /api/history/:id` response includes `is_public`
- `PATCH /api/history/:id/visibility` with `{ is_public: false }` returns `{ id, is_public: false }`
- `PATCH /api/history/:id/visibility` by non-owner returns 403
- `PATCH /api/history/:id/visibility` without auth returns 401
- `PATCH /api/history/:id/visibility` with non-boolean body returns 400
