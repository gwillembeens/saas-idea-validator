<name>Create parseScores utility</name>
<action>
Create `client/src/utils/parseResult.js` — extracts numeric scores from Claude's streamed markdown response.

The Claude API returns a markdown scorecard table in this format:
```
| Phase | Score | Weight |
|-------|-------|--------|
| 1. Market & Niche | 4/5 | 30% |
| 2. Content & Distribution | 3/5 | 25% |
| 3. Product & Agent Architecture | 4/5 | 35% |
| 4. Pricing & Moat | 3/5 | 10% |
| **Weighted Total** | **3.7/5** | |
```

The regex `/\|\s*([\d.]+)\/5\s*\|/g` matches every cell containing `X/5`, including the weighted total row. We take only the first 4 matches (the phase scores), then compute the weighted total ourselves to ensure calculation accuracy.

```js
// Returns { phase1: 4, phase2: 3, phase3: 4, phase4: 3, weighted: 3.7 } or null on failure
export function parseScores(markdown) {
  try {
    const rows = markdown.match(/\|\s*([\d.]+)\/5\s*\|/g)
    if (!rows || rows.length < 4) return null
    const scores = rows.slice(0, 4).map(r => parseFloat(r.match(/[\d.]+/)[0]))
    const weighted = +(scores[0] * 0.30 + scores[1] * 0.25 + scores[2] * 0.35 + scores[3] * 0.10).toFixed(1)
    return { phase1: scores[0], phase2: scores[1], phase3: scores[2], phase4: scores[3], weighted }
  } catch {
    return null
  }
}
```

<name>Create Arrow decorative SVG component</name>
<action>
Create `client/src/components/decorative/Arrow.jsx` — a hand-drawn style SVG arrow.

Component signature: `Arrow({ direction = 'right', className = '' })`

The SVG simulates a hand-drawn arrow with a slightly wobbly shaft and arrowhead. Uses stroke (not fill) for a sketchy look. The `direction` prop rotates the SVG to point in different directions.

```jsx
export function Arrow({ direction = 'right', className = '' }) {
  const rotations = {
    right: 0,
    down: 90,
    left: 180,
    up: 270,
  }

  return (
    <svg
      viewBox="0 0 120 60"
      width="120"
      height="60"
      style={{ transform: `rotate(${rotations[direction] || 0}deg)` }}
      className={`hidden md:block ${className}`}
    >
      {/* Arrow shaft — slightly wobbly path */}
      <path
        d="M 10 30 Q 30 28 50 30 Q 70 32 90 30"
        stroke="#2d2d2d"
        strokeWidth="2.5"
        fill="none"
        strokeLinecap="round"
      />
      {/* Arrowhead */}
      <path
        d="M 78 18 L 100 30 L 78 42"
        stroke="#2d2d2d"
        strokeWidth="2.5"
        fill="none"
        strokeLinecap="round"
        strokeLinejoin="round"
      />
    </svg>
  )
}
```
