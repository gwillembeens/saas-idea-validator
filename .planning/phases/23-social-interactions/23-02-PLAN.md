## Overview
Build the `useLike` hook, `LikeButton` and `CommentCount` components, then extend `ProfileValidationCard` and `LeaderboardCard` footers with social counts. This plan depends on 23-01 (backend routes must exist) and is a prerequisite for 23-03 and 23-04.

- Backend returns `like_count` and `comment_count` on leaderboard entries and profile validations

## Tasks

### Task 23-02-00: Create `useLike` Hook
**Goal:** Custom hook managing like state with optimistic updates

**Files:**
- `client/src/hooks/useLike.js` — New file
  - State: `{ liked: boolean, count: number, loading: boolean, error: string|null }`
  - On mount: fetch `GET /api/results/:id/like-status` → set `liked` and `count`
  - `toggleLike()` action:
    - Guard: if not authenticated, call `onAuthRequired()` callback and return
    - Optimistic update: flip `liked`, adjust `count` ±1 immediately
    - Call `POST /api/results/:id/like` via `fetchWithAuth`
    - On success: update with server response `{ liked, count }`
    - On error: revert optimistic update, set error
  - Accept `resultId` and optional `initialLiked` / `initialCount` props (to skip fetch when card already has counts)
  - Export: `{ liked, count, loading, error, toggleLike }`

### Task 23-02-01: Create `LikeButton` Component
**Goal:** Heart button with filled/outline states, compact and prominent modes

**Files:**
- `client/src/components/social/LikeButton.jsx` — New file
  - Props: `resultId`, `initialLiked?`, `initialCount?`, `mode?: 'compact' | 'prominent'` (default `'compact'`), `onAuthRequired?`
  - Uses `useLike(resultId, { initialLiked, initialCount, onAuthRequired })`
  - Renders `lucide-react` `Heart` icon (stroke-width 2.5):
    - Liked: `fill="#ff4d4d" stroke="#ff4d4d"` with `shadow-hardSm` on the button
    - Not liked: `fill="none" stroke` with `text-pencil opacity-50`
  - Count shown inline next to icon
  - `compact` mode: `h-8` button, `text-sm`
  - `prominent` mode: `h-11` button, `text-base`, visible label "Like" / "Liked"
  - Disabled and slightly dimmed when `loading`
  - Transition: `transition-all duration-100` on fill/shadow change
  - If user not authenticated and `onAuthRequired` is set, call it on click; otherwise dispatch click normally (server returns 401 which useLike handles)

### Task 23-02-05: Create `formatDate` Utility
**Goal:** Relative date for <7 days, short absolute for older

**Files:**
- `client/src/utils/formatDate.js` — New file
  - `formatDate(isoString)`:
    - If diff < 1 min: `"just now"`
    - If diff < 1 hour: `"N minutes ago"`
    - If diff < 24 hours: `"N hours ago"`
    - If diff < 7 days: `"N days ago"`
    - Else: `"Mar 24, 2025"` (short month name, no year if current year)
  - Export as named export
