# Plan 23-03: Frontend Wave 2 ‚Äî Comments UI & Result Page Integration

**Phase:** 23 ‚Äî Social Interactions
**Plan:** 03 of 04
**Status:** Ready

## Overview
Build the full comments UI layer: `useComments` hook, `CommentItem`, `CommentList`, `CommentForm`, `CommentsSection` (inline), and `CommentModal`. Then wire `LikeButton` (prominent) and `CommentsSection` into `ResultPage`. This plan depends on 23-01 (API routes) and 23-02 (LikeButton, CommentCount components).

## Prerequisites
- Plan 23-01 complete: `GET/POST /api/results/:id/comments`, `POST /api/comments/:id/replies`, `DELETE /api/comments/:id` routes live
- Plan 23-02 complete: `LikeButton`, `CommentCount`, `useLike`, `formatDate` available

## Tasks

### Task 23-03-00: Create `useComments` Hook
**Goal:** Custom hook for loading, posting, replying, and deleting comments

**Files:**
- `client/src/hooks/useComments.js` ‚Äî New file
  - State: `{ comments: [], loading: boolean, error: string|null, submitting: boolean }`
  - On mount (and when `resultId` changes): fetch `GET /api/results/:id/comments` ‚Üí set `comments`
  - `addComment(body)`: POST `/api/results/:id/comments`, append returned comment to `comments` state
  - `addReply(parentId, body)`: POST `/api/comments/:parentId/replies`, append returned reply to the matching parent's `replies` array in state
  - `deleteComment(id)`: DELETE `/api/comments/:id`, remove comment from state (or remove reply from parent's replies array)
  - All write actions use `fetchWithAuth`
  - Handle 401 ‚Üí call `onAuthRequired()` callback if provided
  - Export: `{ comments, loading, error, submitting, addComment, addReply, deleteComment }`

### Task 23-03-01: Create `CommentItem` Component
**Goal:** Single comment row with author, timestamp, body, reply/delete actions

**Files:**
- `client/src/components/social/CommentItem.jsx` ‚Äî New file
  - Props: `comment: { id, author_username, body, created_at, is_own, replies: [] }`, `onReply?: (id) => void`, `onDelete?: (id) => void`, `currentUserId?: number`
  - Renders:
    - Author: `font-heading text-sm text-pencil` (link to `/profile/:author_username` if username exists)
    - Timestamp: `font-body text-xs text-pencil opacity-50` using `formatDate`
    - Body: `font-body text-sm text-pencil`
    - Actions row: "Reply" button (if `onReply` provided, only for root comments), "Delete" button (only if `is_own === true`)
  - Delete button triggers `onDelete(comment.id)` ‚Äî confirmation handled by parent
  - No expand/collapse for replies ‚Äî always visible

### Task 23-03-03: Create `CommentForm` Component
**Goal:** Textarea with char count and submit button

**Files:**
- `client/src/components/social/CommentForm.jsx` ‚Äî New file
  - Props: `onSubmit: (body) => void`, `submitting: boolean`, `placeholder?: string`, `maxLength?: number` (default 500)
  - State: `body` (local string)
  - Renders:
    - `<TextArea>` component (existing) with `rows={3}` and placeholder
    - Char count: `font-body text-xs text-pencil opacity-50` right-aligned below textarea
