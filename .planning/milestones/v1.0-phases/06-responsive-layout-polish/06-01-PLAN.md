# Phase 6 Plan 01: Page Assembly & Responsive Layout

## Objective

Assemble all built components (IdeaInput, ResultsPanel, Scorecard, VerdictBadge, decorative elements) into a complete, responsive page layout. Replace the Vite scaffold in `App.jsx` with a mobile-first layout that orchestrates the validator flow and applies consistent design system. This is the final assembly before end-to-end testing.

## Context

All UI components and design system tokens are built in phases 4–5. App.jsx currently contains the Vite default scaffold (counter demo, hero images, links). Phase 6 rewires this into the actual validator experience: hero section explaining the tool, input card, results area with scorecard and verdict, decorative elements.

Key design principles from CLAUDE.md:
- Mobile-first, single column; expand to grid at `md:` breakpoint
- All text in pencil (#2d2d2d), never pure black
- All headings use Kalam 700, body text uses Patrick Hand 400
- Section padding always `py-20`
- Wobbly borders and hard shadows applied
- Decorative SVGs (Arrow, Squiggle) hidden on mobile

Completely replace the contents of `client/src/App.jsx` with:

```jsx
import { useSelector } from 'react-redux'
import { AppShell } from './components/layout/AppShell'
import { IdeaInput } from './components/validator/IdeaInput'
import { ResultsPanel } from './components/validator/ResultsPanel'
import { Scorecard } from './components/validator/Scorecard'
import { VerdictBadge } from './components/validator/VerdictBadge'
import { Arrow } from './components/decorative/Arrow'

export default function App() {
  const status = useSelector(s => s.validator.status)
  const result = useSelector(s => s.validator.result)

  return (
    <AppShell>
      <div className="flex flex-col items-center justify-start min-h-screen px-4 py-20 md:px-8">

        {/* Hero Section */}
        <div className="w-full max-w-2xl text-center mb-20 md:mb-24">
          <h1 className="font-heading text-5xl md:text-6xl text-pencil mb-6 leading-tight">
            Validate Your SaaS Idea
          </h1>
          <p className="font-body text-lg md:text-xl text-pencil leading-relaxed">
            Paste your startup idea and get an honest, investor-grade analysis against our
            30-step framework for building agent-native SaaS businesses.
          </p>
        </div>

        {/* Input Section */}
        <div className="w-full max-w-2xl mb-20 md:mb-24">
          <IdeaInput />
        </div>

        {/* Arrow decoration — hidden on mobile */}
        {status !== 'idle' && (
          <div className="hidden md:flex justify-center mb-12">
            <Arrow direction="down" />
          </div>
        )}

        {/* Results Section — only render when validation has started */}
        {status !== 'idle' && (
          <div className="w-full max-w-2xl space-y-8 md:space-y-12">

            {/* Results Panel */}
            <div>
              <ResultsPanel />
            </div>

            {/* Verdict Badge — below results */}
            {result && (
              <div className="flex justify-center">
                <VerdictBadge />
              </div>
            )}

            {/* Scorecard — visual representation */}
            {result && (
              <div>
                <Scorecard />
              </div>
            )}

          </div>
        )}

        {/* Footer spacing */}
        <div className="mt-20 md:mt-24" />

      </div>
    </AppShell>
  )
}
```
</action>
