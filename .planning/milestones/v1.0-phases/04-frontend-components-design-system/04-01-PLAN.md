<task id="04-01-03">
<name>Create AppShell with dot-grid background</name>
<action>
Create `client/src/components/layout/AppShell.jsx` — page wrapper with iconic dot-grid paper background.

From CLAUDE.md Body Background (AppShell) section, the component renders a div with these exact inline styles:
- backgroundColor: '#fdfbf7' (paper color)
- backgroundImage: 'radial-gradient(#e5e0d8 1px, transparent 1px)' (muted dots)
- backgroundSize: '24px 24px' (24 pixel grid repeat)

Plus Tailwind classes for full viewport coverage: 'min-h-screen w-full'

Component accepts children prop. Export as named export `AppShell`.

```jsx
export function AppShell({ children }) {
  return (
    <div
      style={{
        backgroundColor: '#fdfbf7',
        backgroundImage: 'radial-gradient(#e5e0d8 1px, transparent 1px)',
        backgroundSize: '24px 24px',
      }}
      className="min-h-screen w-full"
    >
      {children}
    </div>
  )
}
```

Create the `client/src/components/layout/` directory if it doesn't exist.
</action>
