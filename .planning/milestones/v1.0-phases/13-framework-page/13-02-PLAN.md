# Plan 13-02: Route + Navigation Wiring

## Objective

Wire the `/framework` route into App.jsx, add a Framework link to the navbar for logged-in
users, and add a "See the full 30-step framework →" text link in the HomePage hero section.

## must_haves

- `/framework` route renders `FrameworkPage` (import + Route added to App.jsx)
- Navbar shows "Framework | History | Sign Out" for logged-in users (Framework link before History)
- HomePage hero has text link "See the full 30-step framework →" below the description paragraph
- All links use `<Link to="/framework">` from react-router-dom
- Navbar Framework link: `font-body text-lg text-blue hover:text-accent transition-colors` (same as History link)
- HomePage link: `font-body text-blue hover:text-accent transition-colors`

## Task 1: Add /framework route to App.jsx

<read_first>
- client/src/App.jsx (full file — understand existing imports and Routes structure)
</read_first>

<action>
In `client/src/App.jsx`:

1. Add import after existing page imports:
   ```js
   import { FrameworkPage } from './pages/FrameworkPage'
   ```

2. Add Route inside the `<Routes>` block, after the `/history/:id` route:
   ```jsx
   <Route path="/framework" element={<FrameworkPage />} />
   ```

The Routes block should look like:
```jsx
<Routes>
  <Route path="/" element={<HomePage />} />
  <Route path="/history" element={<HistoryPage />} />
  <Route path="/history/:id" element={<ResultPage />} />
  <Route path="/framework" element={<FrameworkPage />} />
</Routes>
```
</action>

<acceptance_criteria>
- `grep "FrameworkPage" client/src/App.jsx` matches on 2 lines (import + Route)
- `grep "path=\"/framework\"" client/src/App.jsx` matches
- `grep "import { FrameworkPage }" client/src/App.jsx` matches
</acceptance_criteria>
